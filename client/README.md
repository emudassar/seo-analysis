# SEO Pilot (Client)

AI-powered website SEO analyzer. Paste a URL, get scores, keywords, issues, and recommendations.

## Features

- Instant SEO audits with Playwright + Gemini AI
- Dashboard with recent reports
- Full report view with category scores
- Analysis history

## Run locally

```bash
npm install
npm run dev
```

Set `VITE_BACKEND_URL=http://localhost:5000` in `.env`.
